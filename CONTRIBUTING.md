# 贡献指南

感谢你对 Huhb3D Synthetic Data Generator 的关注！欢迎任何形式的贡献。

## 如何贡献

### 报告 Bug

1. 点击 GitHub/Gitee 仓库的 **Issues** 标签
2. 点击 **New Issue**
3. 选择 **Bug Report** 模板
4. 填写以下信息：
   - 操作系统版本（如 Windows 11 23H2）
   - 复现步骤（你做了什么操作）
   - 期望结果 vs 实际结果
   - 错误日志（如果有）

### 提出新功能

1. 点击 **Issues** → **New Issue** → **Feature Request**
2. 描述你希望的功能和使用场景

### 提交代码

1. Fork 本仓库
2. 创建分支：`git checkout -b feature/your-feature-name`
3. 修改代码
4. 提交：`git commit -m "Add: your feature description"`
5. 推送：`git push origin feature/your-feature-name`
6. 创建 Pull Request

### 代码规范

- C++ 代码遵循当前项目风格（看现有代码怎么写就怎么写）
- Python 代码使用 4 空格缩进
- 提交信息用中文或英文均可，但要清晰描述改了什么
